# Task 15 - K-Means Clustering

## Objective

The objective of this task is to perform customer segmentation using the K-Means Clustering algorithm. Customers are grouped based on their Annual Income and Spending Score to identify different customer categories.

---

## Dataset

Dataset Used: Mall Customers Dataset

Features Used:
- Age
- Annual Income (k$)
- Spending Score (1-100)

---

## Tools & Libraries

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

---

## Steps Performed

- Loaded the dataset
- Checked missing and duplicate values
- Performed data visualization
- Selected required features
- Applied StandardScaler
- Used Elbow Method to find the best value of K
- Calculated Silhouette Score
- Applied K-Means Clustering
- Created customer segments
- Visualized the clusters
- Saved the final output file

---

## Results

- Number of Clusters: 5
- Silhouette Score: 0.4166
- Clusters remained stable with different random seeds.

---

## Customer Segments

- Budget Customers
- Young Spenders
- Premium Customers
- Careful Customers
- Average Customers

---

## Output

The project generates:

- Cluster Visualization
- Elbow Method Graph
- Customer Segmentation Output
- CSV File with Customer Types

---

## Conclusion

K-Means clustering was successfully applied to the Mall Customers dataset. Different customer groups were identified based on their purchasing behavior. These segments can help businesses understand customers better and create suitable marketing strategies.
